Start in HQ, do a knee slice.  How? choice is yours.  Resisting opponent.

Reset.

how is that fundamentally different from Ecological? while still being a drill...?

I suppose it's in how tightly you script the solution.
Maybe forcing a knee-slice is not open enough, a back step or floating might be more appropriate.

A pure Constraint-Led could start in HQ, but you'd set a constraint (bottom player only frames, does not grab), and see what solutions emerge.

Position sparring = constrained situation
Constraint-led = constrained affordances, emergent solutions.
